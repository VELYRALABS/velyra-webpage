# PROJECT LYRA - VeLYRA Healthcare

AI-powered vision and generative intelligence platform for hospitals. Healthcare-native architecture with cloud and edge deployments.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Features

- Multi-portal system (Patient, Doctor, Admin, Pharmacy)
- Vision AI & Generative AI
- Smart pharmacy management
- EHR integration
- Diagnostic AI assistant
- Admin intelligence dashboard

## Tech Stack

- Next.js 15+
- React 19
- TypeScript
- Tailwind CSS
- Framer Motion
- Shadcn/ui components

## Project Structure

```
app/              - Next.js app router pages
components/       - React components & UI library
lib/              - Utilities & database schema
public/           - Static assets
```
